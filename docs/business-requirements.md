# 業務仕様

占いの世界では、依頼者にとって縁起のいい色のことを**エネルギーUP色**ということがあります。  
このドキュメントでは、占い師がこのエネルギーUP色の組み合わせを鑑定する際の仕様について記述します。

## 用語の定義

このドキュメント上で頻繁に登場する用語について、以下のように定義します。

| | 用語 | 定義 |
| :- | :- | :- |
| 1 | 占い師 | 占いを行う立場の人 |
| 2 | 依頼者 | 占い師に占ってもらう立場の人 |
| 3 | エネルギーUP色 | 依頼者にとって縁起のいい色 |

## 鑑定の流れ

1. 占い師は事前に依頼者の本名と生年月日をヒアリングします。
   このステップは、占い師と依頼者の間でLINEやメールなどをやり取りすることによって実現されます。
2. 占い師は手順1でヒアリングした情報を基に、
   予め定義されたエネルギーUP色の組み合わせの中から依頼者に該当する組み合わせを照合します。
3. 占い師は依頼者と面会し、手順2の結果特定したエネルギーUP色の組み合わせを依頼者に伝えます。

> [!IMPORTANT]
> 実際には、エネルギーUP色の他にもいくつかの情報を依頼者に伝えます。
> どんな情報を伝えるのか、詳細仕様については後述します。

> [!IMPORTANT]
> 依頼者の本名と生年月日から鑑定結果を導くロジックについては、占い師しか知り得ない情報です。
> 本プロジェクトにおいてはこの情報は必要ないので割愛します。

## 鑑定に必要な情報

[鑑定の流れ](#鑑定の流れ)でも言及しましたが、鑑定に必要な情報は以下の情報です。  
言い換えると、これが鑑定時のインプットです。

1. 依頼者の本名
2. 依頼者の生年月日

## 依頼者に伝える情報

鑑定のアウトプットは以下の通りです。

1. エネルギーが落ちる年
2. エネルギーUP色の組み合わせ
3. タブー色（薬色）
4. 「能力を引き出すインテリア」に関するアドバイス
5. その年のラッキーカラー・タブーカラー

なお、1~4についてはパターンが決まっています。
その点については後述します。

### アウトプットの各項目

#### 1. エネルギーが落ちる年

業務仕様について調査中です。  
また、一定のパターンがあります。

#### 2. エネルギーUP色の組み合わせ

依頼者にとって縁起のいい色を**エネルギーUP色**といいます。  
エネルギーUP色には自分を守ってくれる色である守護色と、それ以外の自分色をはじめとする6種類があります。  
この7種類のエネルギーUP色の組み合わせを依頼者に伝えます。

守護色以外の6種類のエネルギーUP色の説明は以下の通りです。

|    | 名前       | 説明                                     |
| :- | :-         | :-                                       |
| 1  | 自分色     | 自分の魅力を引き出す色                   |
| 2  | やる気色   | やる気が出る色                           |
| 3  | 精神安定色 | 精神を安定させる色                       |
| 4  | 決断色     | 大きな決断をするときに身につけると良い色 |
| 5  | 健康色     | 健康に導く色                             |
| 6  | 経済色     | 金運に関する色                           |

守護色以外の6色の組み合わせには一定のパターンがあります。  
パターンについては後述します。

> [!IMPORTANT]
> 守護色はパターン化されていません。  
> 占い師による自由入力になります。

#### 3. タブー色（薬色）

タブー色は、端的に言えば依頼者にとってのアンラッキーカラーです。  
この色の物の着用を避けることが推奨されます。

タブー色は以下の中から1色が選択されます。

* ブルー
* グリーン
* イエロー
* レッド
* ブラウン
* パープル

タブー色は推奨されない色ですが、エネルギーが落ちる年には少し扱いが変わります。
エネルギーが落ちる年においてはタブー色は薬のように扱われ（薬色）、ラッキーカラーと組み合わせて使用することができます。
この点は鑑定時に占い師が依頼者に説明するので、「タブー色として扱うのか、薬色として扱うのか」については依頼者が自分で判断することになります。

#### 5. その年のラッキーカラー・タブーカラー

依頼者の**その年の**ラッキーカラー、タブーカラーです。  
それぞれに対して1色または2色ずつを依頼者に伝えます。  
なお、これらの色については占い師による自由入力式となっており、一定のパターンは存在しません。
