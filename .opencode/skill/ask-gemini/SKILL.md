# Ask Gemini CLI

Executes the local Gemini CLI (`gemini`) to get coding assistance.

## Quickstart

```bash
bun gemini -p "Your question or task here" \
  --model "gemini-3-flash-preview" \
  --yolo
```

## 注意点

Geminiのモデルには`gemini-3-flash-preview`を必ず指定して下さい。
